Cybersecurity Learning Roadmap

Beginner → Advanced Learning Path

STEP 1
Cybersecurity 101
Learn basics, CIA triad, threats, attacks, and security domains.
STEP 2
Networking
TCP/IP, DNS, ports, protocols, routing and firewalls.
STEP 3
Operating Systems
Linux, Windows, permissions, processes and file systems.
STEP 4
Linux
Linux commands, file system, networking tools and permissions.
STEP 5
Web Fundamentals
HTTP, cookies, sessions and web architecture.
STEP 6
Blue Team
Logs, monitoring, SOC, incident response and detection.
STEP 7
Red Team
Recon, exploitation, web attacks and ethical hacking.
STEP 8
Cloud Security
AWS, Azure, IAM roles, cloud vulnerabilities and security.
STEP 9
Malware Analysis
Static & dynamic malware analysis and reverse engineering.
